This week I learned about the Document Object Model (DOM) and how to select, access, and update elements in the DOM tree using JavaScript functions. For instance, you can select an element using getElementById() by passing in the id of the element and then find its text node through its children and then access the contents of the text node using nodeValue, which you can then use to update it.

Tools used:
The textbook (JavaScript and JQuery: Interactive Front-End Web Development by Jon Duckett)
W3Schools (HTML DOM elements)
